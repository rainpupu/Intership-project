# 猫个体识别训练报告

## 训练配置

| 参数 | 值 |
|---|---|
| 模型 | ResNet50 + ArcFace |
| 特征维度 | 512 |
| 训练轮数 | 100 |
| 批次大小 | 64 |
| 学习率 | 0.0003 |
| 输入尺寸 | 224×224 |

## 数据集

| 项目 | 数量 |
|---|---|
| 训练集猫数量 | 228 |
| 测试集猫数量 | 57 |
| 训练集图片 | 2240 |
| 测试集图片 | 573 |

## 最佳结果

| 指标 | 数值 |
|---|---|
| **Rank-1** | **0.7295** |
| Rank-5 | 0.7784 |
| mAP | 0.4972 |
| 最终 Loss | 3.6860 |

> 注意：测试集中的 57 只猫在训练时**完全未出现过**，Rank-1 反映模型对新猫的泛化能力。

## 输出文件

| 文件 | 说明 |
|---|---|
| `best_model.pth` | 最佳模型权重（基于 Rank-1） |
| `final_model.pth` | 最终 epoch 模型权重 |
| `training_history.json` | 训练历史数据 |
| `loss_curve.png` | Loss 曲线图 |
| `rank_curves.png` | Rank-1 / Rank-5 曲线图 |
| `map_curve.png` | mAP 曲线图 |
| `training_overview.png` | 四合一总览图 |
