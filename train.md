  三、如何在算力云上训练（以 AutoDL 为例）

  推荐流程：在本地/算力云训练 → 导出 .pt → 上传到 VisAgent

  步骤 1：准备数据集

  在算力云实例上准备以下结构：
  /dataset/traffic_light/
  ├── images/
  │   ├── train/     ← 1,781 张（来自 train.json 标注的 1781 张图像）
  │   ├── val/       ← 445 张（来自 eval.json 标注的 445 张图像）
  │   └── test/      ← 45 张（来自 eval/ 目录，无标注，纯测试用）
  ├── labels/
  │   ├── train/     ← 1,781 个 .txt 文件（从 train.json 转换）
  │   └── val/       ← 445 个 .txt 文件（从 eval.json 转换）
  └── data.yaml      ← 数据集配置文件
  
  data.yaml 内容：
  path: /dataset/traffic_light
  train: images/train
  val: images/val

  nc: 4
  names: ['Red', 'Yellow', 'Green', 'Off']

  步骤 2：运行训练脚本

  在算力云实例上执行：
  # 安装依赖
  pip install ultralytics torch==2.2.2

  # 开始训练
  yolo train \
    model=yolo11n.pt \
    data=/dataset/traffic_light/data.yaml \
    epochs=100 \
    batch=16 \
    imgsz=640 \
    lr0=0.01 \
    device=0

  或者编写 Python 脚本 train.py：
  from ultralytics import YOLO
  model = YOLO("yolo11n.pt")
  model.train(
      data="/dataset/traffic_light/data.yaml",
      epochs=100,
      batch=16,
      imgsz=640,
      device=0,
      project="runs/train",
      name="traffic_light_exp",
      patience=20,
  )

  步骤 3：导出并下载模型

  训练完成后，模型保存在：
  runs/train/traffic_light_exp/weights/best.pt   ← 最优模型
  runs/train/traffic_light_exp/weights/last.pt   ← 最后 epoch 模型

  从 AutoDL 实例下载 best.pt 到本地。

  步骤 4：上传到 VisAgent

  通过网页前端 训练管理 → 模型上传，选择场景 traffic_light 和 .pt 文件上传。

  或者直接调 API：
  curl -X POST http://localhost:8888/api/training/models/upload \
    -F "scene_id=5" \
    -F "model_file=@best.pt" \
    -F "model_name=traffic_light_v1" \
    -F "version=v1.0.0" \
    -F "is_default=true"

  之后检测时选择 traffic_light 场景就会自动加载这个模型。

  四、需要什么数据

  对于 traffic_light 场景，需要：

  ┌───────────┬────────────────┬────────────────┬────────────────────────────────────────┐
  │  数据项   │    格式要求    │    数量建议    │                  说明                  │
  ├───────────┼────────────────┼────────────────┼────────────────────────────────────────┤
  │ 训练图像  │ .jpg/.png      │ ≥500张         │ 车载视角拍摄的交通灯照片               │
  ├───────────┼────────────────┼────────────────┼────────────────────────────────────────┤
  │ 验证图像  │ .jpg/.png      │ ≥100张         │ 与训练集不重复                         │
  ├───────────┼────────────────┼────────────────┼────────────────────────────────────────┤
  │ 标注文件  │ YOLO .txt 格式 │ 与图像一一对应 │ 每行: class_id cx cy w h（归一化坐标） │
  ├───────────┼────────────────┼────────────────┼────────────────────────────────────────┤
  │ data.yaml │ YAML           │ 1个            │ 路径 + 类别定义                        │
  └───────────┴────────────────┴────────────────┴────────────────────────────────────────┘

  交通灯类别映射：
  class_id 0 → Red    (红灯)
  class_id 1 → Yellow (黄灯)
  class_id 2 → Green  (绿灯)
  class_id 3 → Off    (未亮灯/灭灯)

  标注示例（一行一个目标）：
  0 0.5 0.3 0.1 0.15    ← 图像中心位置有一个红灯
  2 0.7 0.4 0.08 0.12   ← 右侧有一个绿灯

  数据集推荐来源：
  - LISA Traffic Light Dataset (https://www.kaggle.com/datasets/mbornoe/lisa-traffic-light-dataset)
  - Bosch Small Traffic Lights Dataset (https://huggingface.co/datasets/keremberke/road-traffic-object-detection)
  - 自行采集车载视频 → 截帧 → 标注（LabelImg / LabelMe / CVAT）
